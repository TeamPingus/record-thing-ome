# Record thing

Thing to call api of Ovenmediaengine, to start/stop recording and check status


### Build instructions:
```bash
cargo b -r
./target/release/record-thing-ome -h
```

### Usage:
```bash
./record-thing-ome -h
```

> [!NOTE]
> Make sure, that you have a .env file, or environment variables exported. Refer to ```.env.example```
